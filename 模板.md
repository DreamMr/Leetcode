## 第一个小于等于k的数
```
def findFirstSmallOrEq(self,li,k):
        left = 0
        right = len(li) - 1
        res = left
        while left <=right:
            mid = (left+right) // 2
            if li[mid] > k:
                right = mid-1
            elif li[mid] < k:
                left = mid + 1
                res = mid
            else:
                return mid
        return res
```

## 小于k的斐波那契数列
```
def Feb(k):
    li = [1]
    a,b=1,1
    while b<k:
        c = a+b
        a=b
        b=c
        li.append(c)
    if li[-1] >k:
        li.pop(-1)
    return li
```
